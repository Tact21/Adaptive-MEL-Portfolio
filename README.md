## 📂 Featured Portfolio Projects & Deliverables

### 📊 Project 1: Digital Data Integration & Interactive Visualization (Power BI)
* **Core Output Asset:** `dashboards/Justice_Program_PowerBI_Dashboard.pdf` *(Simulated Dashboard Output)*
* **Description:** A robust data architecture demonstrating how field-level data can be integrated into a centralized format to track justice problem resolution, systemic changes, and gender-disaggregated metrics across fluid multi-regional environments (Oromia, Amhara, and Tigray).
* **Underlying Relational Data Models (Available in `/data`):**
    * `Case Registry`: Transactional ledger mapping individual case IDs, dispute profiles (Land Dispute, Domestic Violence, Human Rights), and custom user satisfaction scores.
    * `Beneficiaries`: Intersectional profile matrix linking demographic cohorts to distinct structural vulnerabilities (e.g., Female-Headed Households, PWDs, Youth).
    * `Training Events`: Tracks capacity development metrics, mapping the correlation between localized legal literacy and dispute resolution velocity.
    * `Feedback Register`: Accountability verification engine capturing real-time inquiries, suggestions, and complaints to measure user-perceived procedural equity.
    * `KPI Dashboard`: Consolidated metric matrix displaying a **94.5% case resolution rate** and a **4.2/5 mean user satisfaction score**.

### 📝 Project 2: Advanced Qualitative Analysis & Participatory Learning (MSC Framework)
* **Core Document Asset:** `MSC_Methodology_Case_Study-v3.pdf`
* **Description:** A strategic framework outlining the operational implementation of the **Most Significant Change (MSC)** methodology. This document details how qualitative "change stories" are systematically captured, audited, and deployed in participatory learning reviews to dynamically adapt Theory of Change (ToC) assumptions in fragile contexts.
* **Retrospective Framework Integration (Methodological Policy Briefs):**
    * **National Electoral Justice (NEBE Hubs):** Shifted monitoring from bureaucratic asset tracking to capture active justice outcomes. Monitored the processing velocity and localized pathways of 1,240 electoral disputes, validating a **34% drop in regional violent escalation**.
    * **Legal Empowerment & CSO Mobilization (45 Hubs):** Rather than simply logging files closed, this maps 2,340 legal aid journeys directly to HiiL's Resolution Pathways. Confirms that 78% of local disputes were resolved safely via community mediation, proving an immense macro-economic optimization (**$12 community mediation mean cost vs. $340 formal court costs**).
